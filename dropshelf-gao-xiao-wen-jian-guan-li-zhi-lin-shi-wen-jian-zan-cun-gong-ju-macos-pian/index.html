Dropshelf - 高效文件管理之临时文件暂存工具 macOS 篇

什么是临时文件暂存

提供一个临时的空间，方便把文件/链接/图片/文本等信息进行暂存，并灵活在多个软件中进行复制/移动等操作。

工具介绍

Dropshelf

简介及使用

{ Mac | 免费[1]/闭源 }
主页: https://pilotmoon.com/dropshelf/

这个是自己购买后一直在使用的软件，软件打开后没有任何界面，把文件拖在屏幕左右边缘后会自动滑出一个临时的搁板（英文：shelf，介绍最多可以有 20 个并未实测）。拖拽后自动收回，鼠标靠近后自动展开，可很方便的从临时搁板把文件拖动到其它软件。

整体设置也比较简单，主要就三项：可以激活的边缘，是否最小化未激活的搁板，内容被拖出去后是否关闭搁板。

当图片、HMTL、链接、纯文本被放入搁板时，右键还会对应一些不同的功能。

PS：这个软件和 PopClip [2] 是同一家公司的产品，整体属于比较有用的效率小工具。

优缺点

优点：
  • 全功能无限制免费使用；
  • 适配了 M1 芯片；
  • 多屏幕的支持；
  • 支持浏览器中的图片以及文字片段等；
  • 整洁的界面并支持明暗模式，毕竟之前是付费软件。
缺点:
  • 语言目前的 beta 版本只支持英文，但是功能和配置都比较简单，不影响使用。

同类软件

Unclutter：{ Mac | 收费/闭源 } 在桌面顶部一个整合了剪贴板、文件和便笺的存储空间。
Yoink：{ Mac/iPad/iPhone | 收费/闭源 } 类似 Dropshelf 提供一个搁板存放各个类型的文件，还支持 iPad 和 iPhone。
Dropzone：{ Mac | 收费/闭源 } 可以快速支持文件的移动、复制，运行应用，上传到云端，触发自动化脚本等。

  2. Mac 下一款选中文字后弹出复制、翻译等增强功能的小工具 ↩︎
