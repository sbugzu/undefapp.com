DropPoint - 高效文件管理之临时文件暂存工具 Windows 篇

什么是临时文件暂存

提供一个临时的空间，方便把文件/链接/图片/文本等信息进行暂存，并灵活在多个软件中进行复制/移动等操作。

工具介绍

DropPoint

简介及使用

{ Windows/Mac/Linux | 免费/开源 }
主页： https://droppoint.netlify.app/
Github: https://github.com/GameGodS3/DropPoint

在 Windows 下找了半天只找到这一个，功能非常非常简单（或者说是简陋），通过快捷键 Shift + Caps Lock （在 MacOS 中使用 Shift + Tab）在鼠标位置创建一个临时实例，支持拖入和拖出文件。
使用过程中有遇到过 bug，拖动文件到 DropPoint 实例时，实例突然消失（实际应该是一直置顶显示），关闭软件重新打开后解决。

优缺点

优点：
  • 跨平台支持；
  • Windows 下支持虚拟桌面。
缺点：
  • 没有任何配置界面，在安装文件里面也没找到有配置文件。作者已经有计划增加设置模块。
  • 快捷键使用了 Caps Lock，会影响大小写，且无法修改；
  • 只支持本地的文件和文件夹，浏览器中的图片、链接等拖出不支持；
  • 稳定性上还有待验证。
