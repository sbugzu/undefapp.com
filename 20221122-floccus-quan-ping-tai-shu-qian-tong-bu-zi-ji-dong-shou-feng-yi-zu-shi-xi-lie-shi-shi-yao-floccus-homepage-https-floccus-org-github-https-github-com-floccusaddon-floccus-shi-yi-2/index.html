Floccus 全平台书签同步工具 - 自己动手丰衣足食系列

Floccus 是什么

Floccus (Homepage | Github) 是一个全平台的跨浏览器跨设备的书签同步工具，且支持使用自建 WebDAV 服务器。

优势

  1. 支持 Google Drive、NextCloud Bookmarks、兼容 WebDAV 协议的服务（如：自建服务器、坚果云 等）
  2. 支持各种浏览器扩展，如：Chrome、Firefox、Edge、Opera、Brave、Vivaldi…… 除了 Safari；同时支持 Android 和 iOS 移动设备
  3. 支持加密存储

安装

浏览器支持

以 ungoogle-chromium 为例进行安装和配置，先下载扩展 chrome web store，其它主流浏览器扩展 Firefox Add-ons、Edge Add-ons

点击扩展图标，选择新增帐号。

第一步，支持三种同步服务：Nextcloud Bookmarks、WebDAV、Google Drive，这里我们因为使用自建服务器所以选择 WebDAV，其它两种模式的配置也类似。

第二步， 配置好对应的 URL、用户名、密码。

特殊的是最后一项 passphrase，书签通过密码加密后进行存储，如果使用其它三方的存储平台强烈建议配置该选项。如果自建服务可以留空，方便备份。

第三步，文件名可以使用默认。

文件格式有 XBEL 和 HTML 两种选择，原计划使用更加通用的 HTML 格式，但是在 iOS 客户端使用过程中同步报错，所以使用了默认的 XBEL。
本地文件夹，因为部分浏览器不支持直接使用根文件夹，所以选择其它书签文件夹。另外还有一个选项叫同步 Tab，不进行本地书签的同步，只同步已经打开的 Tab 页。

其它默认配置，点击同步之后本地数据就完成上传。同步模式也有三种：

  • 合并（Merge）：默认模式，会合并本地和远程服务端数据，如果会多端编辑选择此模式。
  • 拉（Pull Down）：拉取远程服务端数据，覆盖本地修改。本地不修改，只拉取主设备数据选择此模式。
  • 推（Push Up）：推送本地数据，覆盖远程服务端数据。只有一台主设备修改时，主设备选择此模式。

APP 支持

可以通过 Google Play | F-DROID 和 APP Store 分别下载 iOS 和 Android 应用，配置方式和浏览器扩展一致。

各端配置完成，书签就可以联动起来了。
